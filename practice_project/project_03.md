# VideoDownloader

[https://github.com/JeffMony/VideoDownloader.git](https://github.com/JeffMony/VideoDownloader.git)

## 视频下载SDK功能

* 下载整视频，如mp4/mkv/mov/3gp等视频
* 下载HLS，即M3U8视频
* M3U8 视频下载完成，会生成一个本地的local.m3u8文件
* 视频下载完成，可以点击播放视频文件
* 视频下载数据库记录视频下载信息
* 增加视频下载队列
* 增加M3U8合并为MP4的功能
* 可以定制下载视频的标题和封面
* 增加视频下载的group-name
